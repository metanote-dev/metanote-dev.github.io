# Metanote
Almost every emacs user will use org-mode. Org-mode is great and powerful, but for a long time there is no app that can support mode well on mobile. Today, we have developed Metanote  which fully supports org-mode and can bring the same experience as in emacs or even better(thanks to mobile).

We recommend it to you and hoe more people know about it, use it, and support us as we continue to develop it and make it better.

Metanote is an editor for keeping notes, authoring documents, maintaining to-do lists, planning projects, and more.

Everything is plain text and everything is editable in Metanote.

- Support multiple windows, you can browse and edit multiple files at the same time
- Text markup language support
- Support git synchronization
- Support Agenda view
